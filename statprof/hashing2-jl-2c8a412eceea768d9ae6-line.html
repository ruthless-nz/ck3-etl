75 #=
76 `decompose(x)`: non-canonical decomposition of rational values as `num*2^pow/den`.
77
78 The decompose function is the point where rational-valued numeric types that support
79 hashing hook into the hashing protocol. `decompose(x)` should return three integer
80 values `num, pow, den`, such that the value of `x` is mathematically equal to
81
82 num*2^pow/den
83
84 The decomposition need not be canonical in the sense that it just needs to be *some*
85 way to express `x` in this form, not any particular way – with the restriction that
86 `num` and `den` may not share any odd common factors. They may, however, have powers
87 of two in common – the generic hashing code will normalize those as necessary.
88
89 Special values:
90
91 - `x` is zero: `num` should be zero and `den` should have the same sign as `x`
92 - `x` is infinite: `den` should be zero and `num` should have the same sign as `x`
93 - `x` is not a number: `num` and `den` should both be zero
94 =#
