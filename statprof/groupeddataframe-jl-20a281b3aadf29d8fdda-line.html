1 # Implementation note
2 # There are two important design features of GroupedDataFrame
3 # 1. idx, starts, ends and keymap are by default left uninitialized;
4 # they get populated only on demand; this means that every GroupedDataFrame
5 # has lazy_lock field which is used to make sure that two threads concurrently
6 # do not try to create them. The lock should be used in every function that

10 # a GroupedDataFrame they are not copied for efficiency. If in the future
